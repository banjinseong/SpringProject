# 스프링 게시판

<br>

## 🎯 목적
- 간단한 게시판을 스프링 MVC구조로 만들어보기

<br>

## 🔧개발환경 설정
- IDE: Spring Tool Suite(STS) 3.9.18
  
- 빌드도구: Maven
  
- 서버: Apache Tomcat
  
- 데이터베이스: MySQL
  
- 프레임워크 및 라이브러리:
  - Spring
  - MyBatis
